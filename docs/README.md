## What is Valyrian Edge?

Valyrian Edge is an autonomous AI penetration testing platform designed specifically for LLM-powered applications. It uses AI agents to discover, exploit, and report vulnerabilities in chatbots, RAG systems, and agentic AI applications.

## Key Features

| Feature | Description |
|---------|-------------|
| **10 Security Agents** | Complete OWASP LLM Top 10 coverage |
| **Multi-LLM Support** | Anthropic, OpenAI, Ollama |
| **Automated Exploitation** | PoC generation for verified findings |
| **Report Generation** | Markdown, HTML, JSON, SARIF |
| **Browser Automation** | Playwright for web UI testing |
